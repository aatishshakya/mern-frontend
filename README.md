# MERN Stack Developer Technical Test

## Backend API

### Setup

1. Clone the repository:

   ```bash
   git clone git@github.com:aatishshakya/mern-frontend.git

   ```

2. Install dependencies:

   ```bash
   npm install

   ```

3. Update the constants in env
   API_URL = 'your-endpoint'

4. Start the server:

   ```bash
   npm start
   ```
